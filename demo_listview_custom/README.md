# 自绘列表示例

这个子项目展示了如何在Qt中实现自定义绘制的列表控件，通过自定义委托（Delegate）实现复杂的列表项显示效果。

## 功能特点

- 自定义列表项的绘制
- 支持复杂布局的列表项
- 图文混排显示
- 列表项交互效果
- 自定义列表项编辑器
- 高度可定制的显示样式

## 示例内容

- 自定义委托（Delegate）的实现
- 复杂布局列表项的绘制
- 列表项的鼠标和键盘交互处理
- 列表项动画效果
- 自定义编辑器的实现
- 样式表（QSS）的使用

## 使用方法

1. 打开项目
2. 运行demo_listview_custom示例
3. 观察自定义列表项的显示效果
4. 尝试与列表项进行交互
5. 测试列表项的编辑功能

## 依赖

- Qt 5.15或更高版本
- 项目中的utils库
