# QCustomPlot 曲线控件示例

这个子项目展示了如何使用QCustomPlot库进行各种图表绘制。QCustomPlot是一个功能强大的Qt图表绘制库，支持多种类型的图表和绘图选项。

## 功能特点

- 支持多种图表类型（折线图、柱状图、散点图等）
- 支持多坐标轴
- 支持缩放和拖动
- 支持图例
- 支持导出为图片

## 示例内容

- 基本折线图绘制
- 实时数据更新图表
- 多曲线显示
- 坐标轴设置
- 图表交互（点击、拖动、缩放）

## 使用方法

1. 打开项目
2. 运行demo_qcustomplot示例
3. 通过界面上的控件尝试不同的图表设置和选项

## 依赖

- Qt 5.15或更高版本
- QCustomPlot库 (已包含在项目中)
